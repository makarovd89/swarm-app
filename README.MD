# Template for Wildfy-Swarm application
## Run
##### java -jar target/swarm-app-swarm.jar -Dswarm.http.port=8080 -Dswarm.context.path=/swarm-app
##### mvn wildfly-swarm:run
## Test
##### curl -v http://localhost:15080/swarm-app/api/healthcheck
